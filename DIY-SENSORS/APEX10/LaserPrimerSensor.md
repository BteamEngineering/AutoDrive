# Laser Primer Sensor

A DIY fiber-optic primer presence/orientation sensor for the Mark 7 Apex 10 progressive reloading press.

This project uses an RS320-TZ-M3 fiber head, FS-N18 amplifier, and isolated output circuit to detect missing or improperly oriented primers before the shell plate indexes to the next station.

Features:
- Fiber-optic primer detection
- Isolated output for Apex 10 integration
- DIY/open hardware approach
- Custom 3D-printable housings
