# Windows Firmware Update Guide

This guide explains how to update your AutoDrive controller using Windows.

---

## Step 1 – Connect the System

- Ensure MAIN POWER is OFF
- Do NOT power the machine

- Connect Micro-USB:
  - USB → Computer
  - Micro-USB → Control box

- The board powers via USB

---

## Step 2 – Open the Uploader

- Launch the Gross Engineering Software Uploader
- A command window will open

---

## Step 3 – Verify Firmware

- The uploader auto-detects the .HEX file
- Confirm the correct file is shown

---

## Step 4 – Upload

- Press ANY KEY when prompted
- The system will:
  - Detect COM port
  - Upload firmware

- Wait for:
  - SUCCESS or FAILURE message

---

## Step 5 – If There’s an Error

COM Port Not Found:
- Likely driver issue
- Install drivers if needed

Still not working:
- Contact support

---

## Notes

- Do not disconnect during upload
- Keep power OFF during process
